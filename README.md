This repository presents a collection of Machine Learning tasks completed during my internship, focusing on building practical skills and understanding real-world applications of AI technologies.

Task 1: Decision Tree Implementation
Developed and visualized a Decision Tree model using the Iris dataset to perform classification tasks and understand model decision-making.

Task 2: Sentiment Analysis with NLP
Built a sentiment analysis model using Natural Language Processing techniques to classify text data into positive, negative, and neutral categories.

Task 3: Image Classification Model
Created a deep learning-based image classification system using Convolutional Neural Networks (CNNs) to categorize images accurately.

Task 4: Recommendation System
Designed a recommendation system to suggest relevant items based on user preferences using similarity-based techniques.

These tasks demonstrate my practical knowledge of Machine Learning, Natural Language Processing (NLP), and Deep Learning, along with hands-on experience in implementing models, preprocessing data, and evaluating performance on real-world datasets.
